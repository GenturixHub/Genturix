## Description (4000 caracteres máx)

```
GENTURIX es la plataforma líder de seguridad y gestión para condominios y comunidades residenciales cerradas.

SEGURIDAD PRIMERO
Nuestro sistema de emergencias pone la seguridad de tu familia en primer lugar. Con un solo toque, activa el botón de pánico y envía una alerta instantánea con tu ubicación exacta a los guardias y administradores de tu condominio.

CONTROL DE VISITANTES
Olvídate de las llamadas a la caseta. Pre-autoriza a tus visitantes desde tu iPhone y recibe una notificación cuando lleguen. Cada visita queda registrada para tu tranquilidad.

CARACTERÍSTICAS PRINCIPALES:

• Botón de Pánico
  - Alertas instantáneas con geolocalización
  - Tipos de emergencia: médica, seguridad, general
  - Historial completo de incidentes

• Gestión de Visitantes
  - Pre-autorización desde tu dispositivo
  - Códigos QR de acceso
  - Notificaciones de llegada en tiempo real

• Control de Acceso
  - Panel dedicado para guardias
  - Verificación de identidad
  - Registro de vehículos

• Reservas de Amenidades
  - Salones de eventos
  - Canchas deportivas
  - Áreas comunes

• Administración
  - Dashboard con métricas
  - Gestión de usuarios
  - Reportes y auditoría

PARA TODA LA COMUNIDAD
GENTURIX conecta a todos los miembros de tu condominio:
- Residentes: Emergencias, visitantes y reservas
- Guardias: Control de acceso y respuesta a alertas
- Administradores: Gestión completa del condominio

PRECIO JUSTO
Solo $1 USD por usuario al mes. Sin sorpresas ni contratos largos.

PRIVACIDAD Y SEGURIDAD
Tus datos están protegidos con encriptación de nivel bancario. Cumplimos con todas las regulaciones de privacidad.

Descarga GENTURIX y transforma la seguridad de tu comunidad.

Soporte: support@genturix.com
Web: www.genturix.com
```
